## 🔧 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- HTML e CSS
- JavaScript
- Git e Github
- Figma

## 📌 Projeto

O **DevLinks** é um agregador de links para usar como cartão de visitas online.  
Essa é a **versão personalizada por Matheus Henrique Albert**, com meus links e informações atualizadas.

## ✨ Principais Aprendizados

Durante o desenvolvimento desse projeto, aprendi e pratiquei conceitos importantes como:

- Estruturação de páginas com **HTML semântico**
- Estilização e variáveis de cor usando **CSS custom properties**
- Uso de **JavaScript** para alternar temas (dark/light mode)
- Organização do código e boas práticas no **Git e GitHub**
- Criação de um **README** padronizado e documentado
- Publicação do projeto online utilizando o **GitHub Pages** (em processo de configuração)
